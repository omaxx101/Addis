# 🌆 Addis – New Tab Browser Extension

Addis is a clean and aesthetic browser extension that transforms your new tab into a focused, minimal workspace. It combines productivity and inspiration with a live clock, daily quotes, and a simple to-do list — all on top of a dynamic background.

## ✨ Features

- 🕒 Live Clock – Always know the current time  
- 📝 To-Do List – Track your daily tasks  
- 💬 Inspirational Quotes – Stay motivated  
- 🖼️ Dynamic Background – Refreshes everyday
- 🎨 Minimal UI – Clean and distraction-free  

## 🚀 Installation (Run Locally)

### 1. Download the Project

Clone the repository:
```bash
git clone https://github.com/omaxx101/addis.git
```

Or download as a ZIP and extract it.

---

### 2. Open Extensions Page

In Google Chrome, go to:
```
chrome://extensions/
```

---

### 3. Enable Developer Mode

Turn on **Developer Mode** (top right corner).

---

### 4. Load the Extension

- Click **“Load unpacked”**  
- Select the project folder  

---

### 5. Open a New Tab

Your Addis extension should now appear 🎉

---

## 🔄 Updating the Extension

After making changes:

1. Go to `chrome://extensions/`  
2. Click the refresh icon on Addis  
3. Reload your tab
