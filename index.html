Privacy Policy

This Chrome extension does not collect, store, or transmit any personal user data.

Any preferences (such as user name or settings) are stored locally on the user's device using Chrome storage API. This data never leaves the device and is not shared with any external servers.

The extension does not track browsing history, location, or user activity.

No analytics, advertising, or third-party tracking services are used.

Data Safety

All data remains fully on the user's device and can be deleted at any time by removing the extension.
